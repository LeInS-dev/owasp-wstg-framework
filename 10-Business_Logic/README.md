# WSTG-BUSL - Business Logic Testing

## Overview

This module implements OWASP WSTG Business Logic Testing, focusing on business logic flaws and workflow vulnerabilities.

## Tests Included
- WSTG-BUSL-001: Domain Logic Flaws Testing
- WSTG-BUSL-002: Authorization Bypass Testing
- WSTG-BUSL-003: Workflow Flaws Testing
- WSTG-BUSL-004: Parameter Tampering Testing
- WSTG-BUSL-005: Extreme Conditions Testing

## Usage
```bash
python business_logic_tester.py https://target-domain.com
```

## Key Features
- Business logic vulnerability detection
- Authorization bypass testing
- Workflow manipulation testing
- Parameter tampering analysis
- Extreme input condition testing
