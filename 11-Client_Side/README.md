# WSTG-CLNT - Client-Side Testing

## Overview

This module implements OWASP WSTG Client-Side Testing, focusing on client-side vulnerabilities and browser-based attacks.

## Tests Included
- WSTG-CLNT-001: DOM-Based XSS Testing
- WSTG-CLNT-002: JavaScript Execution Testing
- WSTG-CLNT-003: HTML Injection Testing
- WSTG-CLNT-004: Client-Side Resource Manipulation Testing
- WSTG-CLNT-005: CAPTCHA Bypass Testing
- WSTG-CLNT-006: Clickjacking Testing
- WSTG-CLNT-007: Local Storage Injection Testing

## Usage
```bash
python client_side_tester.py https://target-domain.com
```

## Key Features
- DOM XSS vulnerability detection
- JavaScript security analysis
- HTML injection testing
- Resource manipulation testing
- CAPTCHA security testing
- Clickjacking vulnerability detection
- Local storage security testing
