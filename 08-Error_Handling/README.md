# WSTG-ERRH - Error Handling Testing

## Overview

This module implements OWASP WSTG Error Handling Testing, focusing on improper error handling and information disclosure.

## Tests Included
- WSTG-ERRH-001: Error Codes Testing
- WSTG-ERRH-002: Database Error Messages Testing
- WSTG-ERRH-003: Web Server Error Messages Testing
- WSTG-ERRH-004: API Error Handling Testing

## Usage
```bash
python error_handling_tester.py https://target-domain.com
```

## Key Features
- Stack trace detection
- Database error analysis
- Server information disclosure testing
- API error format analysis
