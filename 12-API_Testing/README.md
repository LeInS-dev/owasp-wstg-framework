# WSTG-APIT - API Testing

## Overview

This module implements OWASP WSTG API Testing, focusing on API security vulnerabilities and REST/GraphQL testing.

## Tests Included
- WSTG-APIT-001: API Discovery Testing
- WSTG-APIT-002: API Authentication Testing
- WSTG-APIT-003: API Authorization Testing
- WSTG-APIT-004: API Input Validation Testing
- WSTG-APIT-005: API Rate Limiting Testing
- WSTG-APIT-006: API Error Handling Testing
- WSTG-APIT-007: API CORS Configuration Testing
- WSTG-APIT-008: API Versioning Testing

## Usage
```bash
python api_tester.py https://target-domain.com
```

## Key Features
- API endpoint discovery
- Authentication bypass testing
- Authorization vulnerability detection
- Input validation testing
- Rate limiting analysis
- CORS configuration testing
- API security assessment
