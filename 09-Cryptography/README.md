# WSTG-CRYP - Cryptography Testing

## Overview

This module implements OWASP WSTG Cryptography Testing, focusing on cryptographic weaknesses and implementation flaws.

## Tests Included
- WSTG-CRYP-001: Transport Layer Security Testing
- WSTG-CRYP-002: Cryptographic Implementation Testing
- WSTG-CRYP-003: Random Value Generation Testing
- WSTG-CRYP-004: Password Storage Testing
- WSTG-CRYP-005: Weak Cryptographic Algorithms Testing

## Usage
```bash
python crypto_testing.py https://target-domain.com
```

## Key Features
- HTTPS/TLS configuration analysis
- Cryptographic algorithm detection
- Randomness analysis
- Password storage security testing
- SSL/TLS certificate analysis
