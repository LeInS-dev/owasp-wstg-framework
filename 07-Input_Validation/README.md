# WSTG-INPV - Input Validation Testing

## Overview

This module implements OWASP WSTG Input Validation Testing, focusing on injection vulnerabilities and input validation flaws.

## Tests Included
- WSTG-INPV-002: Reflected XSS Testing
- WSTG-INPV-005: SQL Injection Testing  
- WSTG-INPV-012: Command Injection Testing
- WSTG-INPV-017: File Inclusion Testing
- WSTG-INPV-022: HTTP Parameter Pollution

## Usage
```bash
python input_validation_tester.py https://target-domain.com
```

## Key Features
- XSS vulnerability detection
- SQL injection testing
- Command injection analysis
- File inclusion vulnerability testing
- Parameter pollution detection
